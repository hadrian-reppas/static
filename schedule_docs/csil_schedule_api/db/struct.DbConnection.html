Struct csil_schedule_api::db::DbConnection

source · 
pub struct DbConnection(PgConnection);
Expand description

This struct represents a connection to the database

A single DbConnection is created at startup and gets shared behind a tokio::sync::Mutex. Right now, the only supported operations are gets, inserts and updates. Because we don’t support more complicated queries, we sometimes make more round trips to the database than we need. Maybe consider adding more functions.

Using an async connection makes no difference because we share the connection between threads anyway.
